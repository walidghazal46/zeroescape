# ZeroEscape No.1

تطبيق تركيز رقمي ورقابة أبوية بواجهة عربية حديثة

## المتطلبات

- Node.js (الإصدار 16 أو أحدث)
- pnpm (مدير الحزم)

## التثبيت

1. فك ضغط الملف
2. افتح Terminal/Command Prompt في مجلد المشروع
3. قم بتثبيت الحزم:

```bash
pnpm install
```

## تشغيل المشروع

```bash
pnpm run dev
```

سيعمل التطبيق على المنفذ المحدد تلقائياً

## بناء المشروع للإنتاج

```bash
pnpm run build
```

## الشاشات المتوفرة

1. **Splash Screen** - شاشة البداية
2. **Login** - تسجيل الدخول
3. **Sign Up** - إنشاء حساب
4. **Onboarding** - التعريف بالتطبيق
5. **Permissions** - إعداد الأذونات
6. **Home Dashboard** - الصفحة الرئيسية
7. **Start Session** - بدء جلسة تركيز
8. **Blocked Apps** - إدارة التطبيقات المحظورة
9. **Web Protection** - حماية الويب
10. **Active Session** - الجلسة النشطة
11. **Block Intercept** - شاشة الحظر
12. **Restart Recovery** - استعادة الجلسة
13. **Emergency Exit** - خروج طارئ
14. **Session Complete** - إتمام الجلسة
15. **Statistics** - الإحصائيات
16. **Settings** - الإعدادات

## التقنيات المستخدمة

- React 18.3
- TypeScript
- Tailwind CSS v4
- React Router
- Recharts (للرسوم البيانية)
- Lucide React (للأيقونات)
- Cairo Font (الخط العربي)

## الميزات

- واجهة عربية بالكامل (RTL)
- تصميم داكن حديث
- جلسات تركيز (دراسة، عمل، نوم، مخصص)
- حظر التطبيقات المشتتة
- حماية الويب DNS
- إحصائيات مفصلة
- استعادة الجلسة بعد إعادة التشغيل

## ملاحظات

- المشروع مبني باستخدام Vite
- جميع المسارات تستخدم React Router
- التطبيق متجاوب ومناسب للهواتف المحمولة
